# XO Game - Tic Tac Toe (C# WinForms)  
# لعبة XO (إكس أو) باستخدام C# و Windows Forms

---

🧩 **Simple XO Game | لعبة XO بسيطة**  
A classic 2-player Tic Tac Toe game built using C# Windows Forms.  
لعبة XO كلاسيكية لشخصين تم تطويرها باستخدام C# و Windows Forms.

---

## ⚙️ Features | المميزات

- 🎮 Two-player mode (X and O)  
  🎮 وضع لاعبين (X و O)

- 🔁 Turn-based system  
  🔁 نظام تبادل الأدوار بين اللاعبين

- 🏆 Win detection (rows, columns, diagonals)  
  🏆 اكتشاف الفائز (صفوف، أعمدة، أقطار)

- 🤝 Draw detection  
  🤝 اكتشاف حالة التعادل

- 🔄 Restart game option  
  🔄 إمكانية إعادة تشغيل اللعبة

- 🎨 Visual highlighting for winning cells  
  🎨 تمييز الخلايا عند الفوز

- 🧱 Built using Windows Forms UI  
  🧱 واجهة رسومية باستخدام Windows Forms

---

## ▶️ How to Run | طريقة التشغيل

- Open the project in **Visual Studio**  
  افتح المشروع باستخدام Visual Studio

- Build the solution  
  قم بعمل Build للمشروع

- Run the program (Start / F5)  
  شغّل البرنامج

- Click on the grid to play  
  اضغط على الخلايا للعب

---

## 🧠 How it works | طريقة العمل

- Player 1 uses X  
  اللاعب الأول يستخدم X

- Player 2 uses O  
  اللاعب الثاني يستخدم O

- Each player takes a turn  
  كل لاعب يلعب بدوره

- First to align 3 symbols wins  
  أول لاعب يحقق 3 رموز يفوز

- If grid is full → Draw  
  إذا امتلأت الخلايا بدون فائز → تعادل
